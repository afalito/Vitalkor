Instrucciones para Favicon de Vitalkor

Archivos necesarios

Para una implementación completa de favicon, se recomienda generar los siguientes archivos:

  • favicon.ico - 16x16, 32x32 y 48x48 pixels
  • favicon.svg - Vector SVG para mejor escalabilidad
  • apple-touch-icon.png - 180x180 pixels
  • android-chrome-192x192.png - 192x192 pixels
  • android-chrome-512x512.png - 512x512 pixels
  • favicon-16x16.png - 16x16 pixels
  • favicon-32x32.png - 32x32 pixels

Implementación

Por el momento, hemos creado:

  • Un favicon.svg con un diseño básico de huella de mascota en color corporativo
  • Un archivo site.webmanifest para configuración básica

Para completar la implementación, se recomienda:

  1. Usar una herramienta como realfavicongenerator.net para generar todos los tamaños necesarios a partir del svg proporcionado
  2. Subir los archivos generados a la carpeta /favicon/
  3. Ya hemos implementado los enlaces necesarios en el <head> del HTML

Código implementado en index.html

Se ha agregado el siguiente código al <head> del HTML:

<link rel="apple-touch-icon" sizes="180x180" href="/favicon/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon/favicon-16x16.png">
<link rel="icon" type="image/svg+xml" href="/favicon/favicon.svg">
<link rel="manifest" href="/favicon/site.webmanifest">
<meta name="theme-color" content="#00b2a9">

Información de color
